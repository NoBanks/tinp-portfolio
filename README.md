## What it is

This Is Not Photography (TINP) is the agentic cinematic gallery for image work that doesn't fit the photography category anymore. The capture step is no longer the camera shutter; it's the prompt, the model, the iterative refinement loop, and the curatorial selection. TINP treats that whole pipeline as the medium.

The cathedral framing is the positioning: when a new medium emerges, the first wave of work that takes it seriously builds the cathedrals. TINP exists to be that cathedral for AI-mediated image work, not a thumbnail mill.
